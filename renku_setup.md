# RENKUのセットアップ方法

## プロジェクトを探す

1. RENKUにアクセスする: [https://renkulab.io/](https://renkulab.io/)
2. GitHubアカウントでログインする
3. 検索窓から「commandLine_2025」を検索する
4. CommandLine_2025を選んでクリックする
   

## セッションを開始する

1. Launch横のプルダウンボタン（V）をクリック
2. "Custom launch"をクリック
3. 下記の通り選ぶ:
    - small
      - 1 CPU, 4GB RAM, 5GB Disk, 0 GPU
    - strageを１GBにする
4. "Continue"をクリックする
5. Rstudioのようなスタートページが表示される
6. コマンドを使うために、"terminal"をクリックする
